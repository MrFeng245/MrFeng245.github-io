下班：Ctrl+S，Alt+F4

快捷键

图层

  • 调出图层面板——F7

  • 新建

    • 按钮

    • Alt+鼠标右键（可以修改属性）

    • Ctrl+Alt+N

  • 选择

    • Ctrl+鼠标左键

    • Shift+鼠标左键

  • 复制

    • Alt+鼠标拖拽

    • Ctrl+G（实则是新建）

  • 移动图层

    • 向下移动：ctrl+[

    • 向上移动：ctrl+]

  • 删除

    • 按钮

    • 拖拽

    • delete键

  • 图层自动选择

    • ctrl

缩放

  • 抓手工具：space

  • 缩放

    • Alt+鼠标滚轮

    • z

    • Ctrl+加号键/减号键

  • 旋转：r+鼠标

移动

  • 自由移动：v+鼠标拖拽

  • 直线方向移动：shift+鼠标拖拽

  • 微调：键盘上下左右（shift加速）

选区

  • 自由：m

  • 正方形：点击后按shift

  • 从中心建立自由：点击后按Alt

  • 取消选区：ctrl+D

  • 重新选择：ctrl+shift+D

  • 隐藏选区：ctrl+H

  • 反向选区：ctrl+shift+I

预备知识

  • 缩放

    • 缩放方式：左下调节百分比

    • 缩放实现

      • 缩放模式下，按住鼠标细微缩放

      • 非缩放模式下，框选需要放大的区域

      • 双击缩放工具，实现100%放大

    • 视图里的导航器

基础

移动

  • 文件a的图层移动到文件b

    • 选择文件a的图层

    • 鼠标拖拽至文件栏

    • 当文件b状态激活（即屏幕显示文件b的图层）

    • 将鼠标拖至合适位置（此时会看见鼠标旁边的小方框）

    • （按住shift可以使文件保持原位）松手，图层复制完成

  • 显示变换控件（选择模式下）

    • 可以对图像进行拉伸旋转

选区

选区知识

  • 保护选区外的区域不受影响

  • 选区工作面板

    • 新选区：新选区替换旧选区

    • 添加到新选区：新选区与旧选区与（shift）

    • 从选区减去：旧选区减新选区（alt）

    • 与选区交叉：新选区与旧选区的交集（shift+alt）

  • 羽化

    • 使边缘区柔和过渡（shift+F6）边缘光滑

  • 样式

    • 正常

    • 固定比例

    • 固定大小

  • 堆叠工具

    • 长按工具栏的选框工具

    • shift+M切换工具

  • 单行、单列：一个像素宽度

建立选区

  • 套索

    • 堆叠工具

      • 多边形选区 #不精确的直线边缘

        • 点选

        • 选择闭合后才能着手其他工作（此时可以ctrl+加号放大）

        • esc取消

        • delete或Backspace回退一个点

      • 磁性套索

        • 自动选区

        • 对比度

          • 边缘清晰：高

          • 边缘模糊：宽度、对比度调低

        • 频率：套索选点速度

  • 选择对象

    • 魔棒

      • 选择颜色相近的区域

      • 取样大小

        • 取样点：单个像素

        • 区域平均值

      • 容差

        • 容差越大，识别范围越大

      • 连续

        • 选择区域从取样点开始辐射

        • 不连续：取样区域全局

    • 快速选择

      • 拉选

      • 画笔

        • 大小（左右中括号）

        • 硬度：边缘识别能力（shift+[]）

        • 间距：连续的识别程度

      • alt键删除部分选区

    • 对象选择

      • 框选

      • 工作栏同上

选区修改

  • 变换选区

    • 右键，变换选区（ctrl+T）

    • esc取消变换

  • 羽化

    • 右键，羽化（shift+F6）

  • 透明度

  • 平滑

    • 倒圆角

  • 扩展

    • 边缘扩展

  • 收缩

    • 边缘收缩

  • 路径（矢量）

    • 像素图变矢量图

打开内联公式设置：文件 -> 偏好设置 -> Markdown -> 内联公式

编辑使用：

  • 行内输入：$ + esc，会出现$$字样，在这两个美元符号之间输入LaTex语法，即可实现在行内插入公式。

  • 独行输入：$$ + 回车，会出现一对$$ $$的符号，在这一对符号之间输入LaTex语法，即可实现在行间插入公式。

如果设置之后还是不能用，那就关闭文件后重新打开即可。

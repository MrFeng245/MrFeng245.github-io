田静英语语法体系

核心：n+v（主语+谓语）

攻略

  • 定位谓语动词，确定句子中包含几件事

  • 定位谓语动词，找到主语

  • 简化句子（去掉介词短语、形容词和副词、限定词），找核心

谓语动词的分类

有实意

  • 不及物动词（intransitive verb）

    • 主语可以自己完成

    • 主谓

  • 及物动词（transitive verb）

    • 主语不能独立完成

    • 主谓宾

    • 主谓双宾（人+物）

    • 主谓宾宾补

    • 注

      • 主谓双宾的宾语可以对调，对掉后需在人前+to/for

      • 主谓双宾——人+物

      • 主谓宾宾补——宾+“是”+宾补 意思完整

无实意

  • （连）系动词
    • 主系表

系动词

包括

  • 系动词列举：条件

  • be动词：单独出现

  • get/become/turn/go/grow：“变得”

  • look/sound/smell/taste/feel：“看、听、闻、尝、感觉起来”

  • seem/appear/keep/remain

用途

  • 简化句子（不及物动词）

  • 找到分裂结构（及物动词）

  • 判断从句

    • 宾语从句——及物动词后

    • 表语从句——系动词后

简单句的核心变化

即谓语动词的变化：

  • 时态

  • 情态

  • 语态

  • 否定

  • 无论谓语动词多么长，依旧记为一个谓语动词

时态

一般时

强调时间，不强调进行/完成

  • 一般过去时

    • 公式：谓语动词 = did

    • 基本含义：过去发生的

  • 一般现在时

    • 公式：谓语动词 = do/does

    • 基本含义：

      • 现在的状态

      • 经常性、习惯性

      • 永恒的（科学事实、客观真理）

  • 一般将来时

    • 公式：谓语动词 =

      • will do

      • am/is/are going to do

    • 基本含义：现在的将来

  • 一般过去将来时

    • 公式：谓语动词 = would do

    • was/were going to do

    • 基本含义：过去的将来

进行时态

强调进行

  • be+doing

  • 过去进行时

    • 公式：谓语动词 = was/were + doing

    • 基本含义：过去正在进行

  • 现在进行时

    • 公式：谓语动词 = am/is/are + doing

    • 基本含义：现在正在进行

  • 将来进行时

    • 公式：谓语动词 =will be doing

    • 基本含义：将来确定进行

完成时态

（强调完成结果，不一定全部完成）在一个时间结点，回头往之前看

  • have+done

  • 注意：by-截止到某一时间

  • 过去完成时

    • 公式：谓语动词 = had done

    • 基本含义：过去的之前（包含过去）【就是老师口中的过去的过去】

  • 现在完成时（**）

    • 公式：谓语动词 = have/has done

    • 基本含义：现在的之前（包含现在）

    • since：过去的时间或动词

    • 常见形式：

      • have/has (词) done

      • 如：has long bothered

      • 词：just(短时间)、ever（曾经）、never（从不）、also（也）

    • 现在完成时和一般过去时

      • 现在全部完成可以与过去时互换

      • 现在部分完成则不能（即老师口中的动作持续到现在，依旧在进行）

  • 将来完成时

    • 公式：谓语动词 = will have done

    • 基本含义：将来的之前

完成进行时

截止到某一时刻，事情进行的过程

  • have been doing

  • 过去完成进行时

    • 公式：谓语动词 = had been doing

    • 基本含义：过去的之前，持续进行

  • 现在完成进行时

    • 公式：谓语动词 = have/has been doing

    • 基本含义：现在的之前，持续进行

  • 一般将来时

    • 公式：谓语动词 = will have been doing

    • 基本含义：将来的之前，持续进行

情态

情绪、态度

特色

  • 情态动词+动词原形

  • 情态动词时态变化有限

  • 无人称变化

  • 变否定/疑问（……can’t……/can he……）

常见

  • 现在与过去一一对应

  • 现在

    • must

    • can

    • will

    • shall

    • many

  • 过去

    • 无

    • could

    • would

    • should

    • might

  • 标红的用法

    • 表过去

    • 表现在（委婉的语气）

    • 表虚拟

作用

  • 情态动词表情态

    • must 必须，强势的要求和命令

    • can/could 能够、可以

    • will/would 将要、愿意

    • may/might 可以、可能

    • should 应该

  • 情态动词表推测

    • must 肯定

    • can’t could’t 不可能

    • 其他：可能

    • 关于时间的推测

      • 情态动词+do：推测现在

      • 情态动词+be doing：推测现在正在

      • 情态动词+have done：推测过去

语态

主动语态

  • 除被动外的所有句子

  • 小知识：英美人士习惯于强调事件的客观性，因此常常使用被动将事件放在前面

被动语态

  • be+done

  • 三要素

    • done：动作

    • be：时间

    • 主语的单复数（决定be的形式）

  • 强调被谁做：be done by 人

  • 被动加时态（她的呼吁被认为是失败的）

    • 过去

      • Her appeal was considered a failure

    • 现在

      • Her appeal is considered a failure

    • 现在的将来

      • Her appeal will be considered a failure

    • 过去的将来

      • Her appeal would be considered a failure

    • 过去进行

      • Her appeal was being considered a failure

    • 现在进行

      • Her appeal is being considered a failure

    • 过去完成

      • Her appeal had been considered a failure

    • 现在完成

      • Her appeal has been considered a failure

  • 判断是不是被动

    • 先看动作

      • 如果是done：主动的完成时/被动/一般过去时

    • 有没有时间be

      • 没有：一般过去时

    • 时间是不是have

      • 是，完成时

      • 否则是被动

  • 被动+情态

    • 情态动词+be+done

否定

  • 实意动词

    • do/does/did + not + v原形

  • 助动词/情态动词

    • 助动词/情态动词 + not

    • （be动词、情态动词、完成时态中的have）

  • 变否定优先边第一个

主宾表的变化（分类）

主语的变化

  • n/代词

  • 非谓语v

    • doing

    • to do

  • 并列多个

表语的变化

  • 4+

  • a.

  • 介词短语

非谓语动词

  • 做主语

    • doing +（something）做主语，使用adv、介词短语、adj修饰

    • to do 做主语一般放在后面

      • It is adj (for sb) to do sth 对某人来说做某事是……的

      • It takes (sb) some time to do sth 做某事花费某人多长时间

  • 做宾语

    • doing

      • 介词后——介宾

      • 动词后——动宾

    • to do

      • 只能放在动词后

      • 宾语太长放则宾补后面

简单句的扩展

词性的扩展

限定词

  • 冠词

  • 数词

  • 名词所有格

  • 名词

形容词或副词

  • 基本作用

    • 形容词或副词修饰成分

      • adj + n

      • v + adv

    • 形容词修饰名词

      • 作定语（前置定语、后置定语）

      • n + 系动词 + adj

    • 副词修饰

      • v

      • 句子

      • 其他形容词

      • 副词

      • 修饰名词做定语，修饰其他做状语

  • 形容词副词的三种比较级

    • 原级比较：as + adj/adv +as + 比较对象

    • 比较级：adj/adv (+ than + 比较对象) 更……

    • 最高级：adj/adv的最高级 + (介词短语表范围)

介词短语

  • 作用

    • 接 n

    • 代词

    • doing 构成介词短语

  • 表时间

    • at/on/in 在时间点/时间上（天）/在时间内（大范围）

    • before/after 之前/之后

    • from/since 自从……时间

    • by/until 直到……时间

    • for 持续……时间

    • during 在……期间

  • 表地点

    • at/in 在……地点/在……里面

      • at 零维（点）

      • on 二维（平面）

      • in 三维（空间）

    • on, above, over/under, blow 在……上面/下面

      • on 有接触

      • over 悬空、垂直

      • above 悬空、斜上

    • in front of/behind 在……之前/之后

    • near/by/beside 在……旁边

    • between, among 在……之间

      • between 两者

      • among >= 3

    • around 在……周围

    • into/out of 进入……里面/从……走出来

      • in 一直在里面

      • into 朝着里面走

    • onto/off 在……上面/脱离……

    • across, through/past 穿过……/经过……

    • along 沿着

    • from/to 来自……/朝着……去

  • 其他

    • of ……的……

      • a of b b的a（ab都是动词，c的b的a也是同理）

    • about 关于

    • with/without 伴随着……/没有……

    • for 为了、因为、对于

    • by 通过……方式、被……做（被动语态）、相差了……（表变化的差额）

      • 之前提到的：地点by——在……旁边、时间by——直到……时间

    • as 作为……、像……一样

    • like 像……一样

    • despite 尽管

      • in spite of = despite

成分的扩展

非谓语动词做定语状语

  • 说明

    • 简单句只能有一个谓语，因此句子中已有谓语时，其他动词只能做非谓语

  • 非谓语(单独出现且构成时态或语态)

    • doing

    • done

    • to do

  • 用法

    • 相当于n，做主/宾/表 to do/doing

    • 修饰n/句子，做定语/状语

  • 非谓语动词的含义

    • doing 主动，正在进行

    • to do 目的，将要去做

    • done 被动，已完成

  • 做定状的时态

    • 非谓语动词做定语，修饰名词，就近原则

    • 非谓语动词做状语，修饰句子，参照句子主语

  • 连词

    • 连接句子

    • 连接词组 突出强调非谓语动词表达的逻辑关系

  • 补

    • do/does/did + v 强调动作

  • 非谓语动词攻略

    • 找全非谓语动词词组

    • 找到非谓语动词所修饰成分的位置，带入含义

    • 知道非谓语动词表达的含义

    • 如果非谓语动词做主宾表，为句子核心，保留，做定状，可以先去掉

同位语

  • 含义

    • 解释说明前面的名词，且与前面的名词所描述同一对象

  • 构成

    • 可以是词、词组、句子

  • 标志性标点

    • 冒号、破折号、两个逗号中间

插入语

  • 含义

    • 补充说明、修饰限定

  • 构成

    • 词、词组、句子

  • 标点

    • 成双的括号、成双的破折号、成双的逗号（这个不绝对）

  • 插入语放句首

    • 主谓结构做插入语

    • 副词做插入语

    • 介词短语做插入语

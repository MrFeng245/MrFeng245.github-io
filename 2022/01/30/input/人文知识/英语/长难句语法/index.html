并列句

并列连词（两个句子中间）

  • 表顺接

    • ……and……和

    • both……and……两者都

    • not only……but……as well 不但……而且（x3）

    • not only…… but also……

    • not only…… but……

  • 表转折

    • ……but ……但是（x2）

    • ……yet……

  • 表选择

    • ……or……或者，二选一（x2）

    • ……either……or

    • neither…… nor……既不……也不

  • 表因果

    • ……for……因为

    • ……so……所以

并列句的省略

  • 相同部分

  • 主语、主谓、主谓宾

攻略

  • 找到省略部分

    • 看时态

    • 看词性

名词性从句

从句相当于名词

写法基本相同，位置不同

主语从句

  • 含义

    • 一个句子做另一个句子主语

  • 写法

    • 同宾语从句

    • 区别

      • that不能省

      • 表是否时只能用whether

  • 位置

    • 句尾，it做形式主语

    • 句首

  • 句型

    • It is done + 主语从句

      • 表达人们对一件事的观点

      • It is said/announced/thought that …

    • It is adj./n. + 主语从句

      • 表达人们对一件事的评价

      • It is a good thing…

      • It is surprising…

宾语从句

  • 含义

    • 一个句子做另一个句子的宾语

  • 从句的变化

    • 陈述句变宾语从句——直接加

      • that：主从句分隔符，可以省

    • 特殊疑问句变宾语从句

      • 先调整语序：主句放在谓语动词前

      • 谓语动词合并（时态合并）

    • 一般疑问句变宾语从句

      • 调整语序，主句提前

      • 加分隔符，if/whether

  • 写法

    • 主句

    • 宾语从句

      • 连接词

      • 陈述句

  • 位置

    • 动宾

      • 动单宾，及物动词+从句

      • 动双宾：及物动词+人+从句

    • 介宾

    • 形容词+宾语

    • 宾语从句后置，it做形式宾语

表语从句

  • 含义

    • 一个句子做另一个句子表语

  • 写法

    • 同宾语从句，that不能省

  • 位置

    • 系动词后面

同位语从句

  • 引导词

    • that

    • where

    • why

    • when

  • 含义

    • 一个句子解释说明抽象名词

  • 写法

    • 同宾语从句

    • that不能省

  • 位置

    • 抽象名词后

    • fact, point, belief, faith, claim, report, worry

名词性从句位置一览

  • 宾语从句（3）

    • 动宾

      • 及物动词+从句

      • 及物动词+人+从句

    • 介宾

    • 形容词+从句

  • 表语从句（1）

  • 主语从句（2）

    • 句首

    • 句尾（it做形式主语）

  • 同位语从句（1）

    • 抽象名词后

攻略

  • 找到从句

    • 开始位置：连接词/分隔符

    • 结束位置：标点、下一个连接词、第二个谓语动词之前

  • 判断从句

    • 从句位置往前看，就近原则

定语从句

概述

  • 定语：修饰限定名词

  • 相同部分替换

    • 人——who/that

    • 物——which/that

  • 先行词 = 关系词

写法

  • 主句（含先行词）

  • 从句

    • 关系词

    • 陈述句

  • 先行词

    • 事/物——which/that

    • 人——who/whom/that

    • 人/物 的——whose

    • 时间——when

    • 地点——where

    • 原因——why

  • 注

    • 定语从句的所有关系词都在从句中做成分

    • whom指人，只能做宾语

    • 关系词在宾语从句（动宾/介宾）时可以省略

分类

  • 限制性

    • 有逗号

    • 因为多个，所以限定

  • 非限制性

    • 无逗号

    • 唯一一个或是相对明确

  • 限制性定语从句与同位语的区别

    • 相同点

      • n + 从句

    • 不同

      • 同位语从句

        • 解释说明n（what）

        • 抽象n后

        • n + that，that不做成分

        • 关系词不能省

      • 限制性定语从句

        • 修饰限定n（which）

        • 任意n后

        • n + that，that做成分

        • n + 省略关系的从句

  • 非限制性定语从句（修饰整句话）

    • 写法

      • 【，】+【which/as】

      • which做主语时不能省

      • 做宾语时可以省

  • 介词提前的限制性定语从句：介词 + whom/which

攻略

  • 不必纠结从句的种类

  • 非限制性定语从句可以省略不看

  • 找到定语从句和先行词

状语从句

概述

  • 描述物体

  • 写法

    • 主语

    • 状语从句

      • 从属连词

      • 完整的陈述句

  • 位置

    • 前、中、后均可

分类

  • 时间状从

    • when/while/as 当……

    • before/after ……之后

    • since/until 直到……

    • as soon as 一……就

    • by the time 截止到

  • 地点状从

    • 及物动词 + where 动宾

    • be/系动词 + where 表从

    • 地点名词/一个和它不搭的词 + where 地点状从

  • 原因状从

    • because

    • since

    • as

  • 结果状从

    • so……that如此……以至于 so + adj./n. + that

    • such ……that

    • so that 所以

  • 目的状从

    • so that 为了

    • in order that

  • 条件状从

    • if 如果

    • once 一旦

    • as /so long as 只要

    • unless 除非，if not 除非不

  • 让步状从（明让步，暗转折）

    • although/though 尽管

    • even if / even though

    • while, however

  • 比较状从

    • than 比较级

    • as 原级

  • 方式状从

    • as 按照

    • as if 似乎

    • as

      • 当……时候，时间状语从句

      • 因为，原因状语从句

      • 和……相比，比较状语从句

      • 按照……的方式，方式状语从句

攻略

  • 完形填空，根据意思选择从属连词

  • 根据从属连词，判断上下文

虚拟句

  • 谓语动词的特殊形式：改变时态

  • if虚拟条件句，真实条件句不用虚拟

    • 假设现在

      • 从句往前推，主句4+1

      • if从句——一般过去时：did、were

      • 主句——could/would/should/might + do

      • If I were you, I would marry him

    • 假设过去

      • if从句——过去完成时：had done

      • 主句——could/would/should/might + have done

    • 假设将来（将来同现在）

      • if主句——did(were), were to do, should do

      • 主句——could/would/should/might + do

倒装句

  • 倒装原因

    • 突出强调

    • 避免头重脚轻

  • 全部倒装——谓语动词全部提前

    • 能看懂即可

    • 突出 a./ ad. /介词短语

    • There be 句型：There be + n　有

      • 值客观存在，不指拥有

  • 部分倒装——变成一般疑问句

    • 会写

    • 否定副词或词组位于句首

    • only位于句首

    • 虚拟条件句可省略if

强调句

  • 构成

    • It is 强调部分 that 剩余部分

  • 可强调内容

    • 词

    • 词组

    • 从句（形容词动词不能强调）

  • 强调句的变化

    • It is/was … that/who …

    • is/was：现在或将来

    • that：人、物均可

    • who：只能是人

  • 区分主从和强调句

    • 去掉 It is/was that，剩余句子完整，强调句

    • a./v. 在 It is/was that 之间，主从句
